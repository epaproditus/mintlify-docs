This page is the canonical reference for every configuration variable used in self-hosted and BYO Connector deployments. iOS build settings live in `Local.xcconfig`; runner settings live in `runner/.env`. Neither file should ever be committed to version control.

## iOS configuration

`Local.xcconfig` overrides the defaults in `Base.xcconfig`. Create it by copying `ios/doit/Config/Local.example.xcconfig` and filling in the values below.

| Variable | Description |
|---|---|
| `DOIT_SUPABASE_URL` | Your Supabase project URL (e.g., `https://xyz.supabase.co`) |
| `DOIT_SUPABASE_ANON_KEY` | Public Supabase anon key (RLS-scoped, safe to embed in the app binary) |
| `DOIT_WAITLIST_URL` | URL displayed to users who open the app without a valid invite code |
| `DOIT_DEVELOPMENT_TEAM` | Apple Developer team ID used for code signing |
| `DOIT_BUNDLE_ID` | Main app bundle identifier (e.g., `com.yourcompany.hermes`) |
| `DOIT_TESTS_BUNDLE_ID` | Bundle identifier for the unit test target |
| `DOIT_UITESTS_BUNDLE_ID` | Bundle identifier for the UI test target |
| `DOIT_WIDGET_BUNDLE_ID` | Live Activity widget bundle identifier (e.g., `com.yourcompany.hermes.widget`) |

## Runner environment variables

Copy `runner/.env.example` to `runner/.env` and populate the sections below. Variables marked **Required** must be set for the runner to start.

### Required

| Variable | Description |
|---|---|
| `SUPABASE_URL` | Supabase project URL — must match `DOIT_SUPABASE_URL` in the iOS config |
| `SUPABASE_SERVICE_ROLE_KEY` | Server-side Supabase key that bypasses Row Level Security — keep this secret |

### Model providers

| Variable | Description |
|---|---|
| `OPENROUTER_API_KEY` | Primary model provider key — used for all Hermes agent runs |
| `OPENAI_API_KEY` | Optional; used for memory extraction if configured |
| `HERMES_MODEL_DEFAULT` | Default model slug passed to Hermes (e.g., `google/gemini-2.5-flash`) |
| `HERMES_MODEL_PROVIDER` | Provider name sent to Hermes (default: `openrouter`) |

### APNs (push notifications)

| Variable | Description |
|---|---|
| `APNS_KEY_PATH` | Absolute path to your `.p8` APNs auth key file on the runner machine |
| `APNS_KEY_ID` | 10-character key ID shown in the Apple Developer portal |
| `APNS_TEAM_ID` | Your Apple Developer team ID (also used in `DOIT_DEVELOPMENT_TEAM`) |
| `APNS_TOPIC` | APNs topic — must exactly match your main app bundle ID |
| `APNS_USE_SANDBOX` | Set to `true` for development/TestFlight builds; `false` for App Store production |

### Integrations

| Variable | Description |
|---|---|
| `COMPOSIO_API_KEY` | Required for all OAuth integrations (Gmail, Google Calendar, etc.) |
| `BROWSERBASE_API_KEY` | Required for web automation tasks |
| `BROWSERBASE_PROJECT_ID` | Browserbase project ID associated with your API key |

### Concurrency and timeouts

| Variable | Default | Description |
|---|---|---|
| `MAX_CONCURRENT_RUNS` | `8` | Maximum number of task executions running simultaneously across all users |
| `MAX_RUNS_PER_USER` | `2` | Maximum number of simultaneous tasks for a single user |
| `RUN_TIMEOUT_SECS` | `900` | Maximum wall-clock time allowed per task execution (15 minutes) |
| `POLL_INTERVAL_SECS` | `2` | How often (in seconds) the runner polls Supabase for new work |

### BYO Connector mode

These variables are only used when `BYO_CONNECTOR_MODE=true`. In this mode the runner forwards task execution to a remote Hermes gateway identified by the connector token rather than using a locally configured Hermes instance.

| Variable | Description |
|---|---|
| `BYO_CONNECTOR_MODE` | Set to `true` to enable BYO Connector routing |
| `DOIT_CONNECTOR_TOKEN` | Scoped connector token generated by the iOS app during pairing |
| `DOIT_CONNECTOR_USER_ID` | The Supabase user ID associated with the connector token |
| `DOIT_CONNECTOR_HERMES_URL` | URL of the local Hermes gateway on the user's machine (default: `http://127.0.0.1:8643`) |
| `DOIT_CONNECTOR_HERMES_API_KEY` | Bearer key required to authenticate requests to the user's Hermes gateway |
| `DOIT_CONNECTOR_PROFILE_NAME` | Optional Hermes profile name to use for this connector (e.g., `byo-hermes`) |

<Warning>
  Never commit `runner/.env` or `ios/doit/Config/Local.xcconfig` to git. Both files contain secrets that would give anyone who reads them full access to your Supabase project and model provider accounts. Both paths are included in the repository's `.gitignore`, but double-check before pushing to a fork or mirror.
</Warning>
