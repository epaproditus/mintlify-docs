Tasks are the core unit of work in Hermes Agent. Instead of navigating menus or filling out forms, you simply describe what you want in plain language — the agent figures out the rest. Whether you're sending an email, scheduling a meeting, or researching a topic, every action starts with a task.

## Input methods

You can create a task three ways: typing, speaking, or attaching a photo. All three feed into the same task composer, and you can mix them freely.

<Tabs>
  <Tab title="Text">
    Tap the **+** button in the bottom-right corner of the main screen to open the composer. Type your task in the text field, then tap **Save**.

    The text field accepts free-form instructions in everyday language. Some examples to get you started:

    - `Send an email to Marcus letting him know the design files are ready for review`
    - `Create a calendar invite for a team sync next Tuesday at 2 PM`
    - `Remind me to submit my expense report before end of day Friday`
    - `Find the three cheapest direct flights from New York to London in August`

    **Tips for writing good tasks:**

    - Name specific people, files, or services rather than using pronouns like "him" or "it"
    - Include any relevant context the agent can't look up on its own (a date, a tone, a word limit)
    - If the task spans multiple steps, describe the full goal rather than individual steps — let the agent plan
  </Tab>
  <Tab title="Voice">
    Tap the **microphone icon** inside the composer to switch to voice input. Tap and hold the microphone button to record — release when you are done speaking.

    Hermes Agent transcribes your speech using OpenAI Whisper, so it handles natural speech, filler words, and light background noise gracefully. After you finish speaking, the transcript appears in the text field — review it, make any edits you like, and tap **Save**.

    Voice is especially useful for longer, context-heavy tasks where typing would be slow. You can dictate a full paragraph of instructions and the agent will parse it correctly.
  </Tab>
  <Tab title="Photo">
    Tap the **camera icon** or the **photo library icon** in the composer toolbar to attach images. You can attach up to **5 images** per task.

    Photos are great for tasks like:

    - `Summarize this document` — attach a photo of a printed report or whiteboard
    - `Draft a reply to this email` — screenshot an email thread and attach it
    - `Book the hotel shown in this screenshot` — attach a screenshot from a travel site
    - `Transcribe the text in this image` — attach a photo of a sign, menu, or handwritten note

    The agent reads the images as part of the task context, so you don't have to re-describe what's in them.
  </Tab>
</Tabs>

## Task preparation

After you tap **Save**, the agent immediately runs a preparation pass before executing your task. You don't need to tap anything else — this happens automatically.

During preparation, the agent:

1. **Rewrites the title** into a concise label (e.g., "Email Marcus — design files ready") so the task is easy to find later
2. **Identifies the right integration** — it decides whether to use Gmail, Google Calendar, web search, or another connected service
3. **Writes a preparation summary** — a short plan that outlines what it intends to do
4. **Auto-starts execution** — the agent begins working through the steps as soon as preparation is complete

The transition from preparation to execution is seamless. By the time you navigate to the task detail view, the agent is often already mid-way through the first step.

<Note>
  If your task is ambiguous — for example, it mentions "my manager" but the agent doesn't have that contact saved — the agent may pause to ask a clarifying question before starting. Answer the question in the task thread and the agent will continue automatically.
</Note>

## Suggested topics

Below the composer, Hermes Agent displays a scrollable **topic strip** with common task categories. Tapping a topic loads a set of starter templates so you don't have to write from scratch.

| Topic | Example templates |
|-------|------------------|
| Communication | Draft an email, send a Slack message |
| Scheduling | Book a meeting, set a reminder |
| Research | Summarize a topic, compare options |
| Documents | Summarize a file, draft a report |
| Coding | Write a script, explain an error |
| Finance | Track an expense, summarize a statement |
| Shopping | Find the best price, compare products |
| Travel | Book a flight, find a hotel |
| Personal | Journal entry, goal tracker |
| Work | Prepare a standup, write a status update |

Tapping any template populates the composer with a pre-written prompt that you can edit before saving.

## Writing effective tasks

The clearest tasks produce the best results. Keep these principles in mind:

- **Be specific about recipients and intent.** Instead of "email him about the project", write "email Daniel about the Q3 roadmap — ask for his feedback by Friday."
- **Include context the agent doesn't have.** If the agent can find it in your inbox or calendar, you don't need to repeat it. But details like tone, deadline, or word count should be spelled out.
- **Use full names for contacts.** The agent can look up "Sarah Chen" in your contacts; "Sarah" alone may be ambiguous.
- **State the outcome, not the steps.** You don't need to say "open Gmail, compose an email, address it to…" — just describe what you want to happen.
- **Let the agent ask.** If you're not sure how to phrase something, write your best attempt. The agent will ask for clarification rather than guess incorrectly.

<Tip>
  **Example of a great task:** "Email my landlord that rent will be 3 days late this month. Be polite, apologize, and ask if there's a partial-payment option. The landlord's address is in my recent emails."

  This task names the recipient implicitly (landlord), specifies the tone (polite), states the goal (ask about partial payment), and tells the agent where to find the missing information (recent emails) — everything it needs to act without interrupting you.
</Tip>

## Task categories

Every task is automatically tagged with a topic (like **Scheduling** or **Communication**) based on what the agent understands about your request. These tags drive two things:

- **Collections** — tasks are grouped into collections in the Recent grid so you can quickly find all your emails, all your research tasks, and so on
- **Topic filter** — tap any topic chip at the top of the Recent grid to filter the list to just that category

You can also manually change a task's topic by opening the task detail, tapping the topic chip, and selecting a new one.
